Design Principles in the Wild

Anthony Rogers

Affordance (click to expand)ign.com
The element of "door usability" is evident on the webpage thanks to the arrows indicating how to change the pictures in the slideshow

Mapping (click to expand)D2L Class Page
The positioning of the check boxes as well as having the progress bar above them indicates that by checking the boxes, the progress bar will increase.

Chunking (click to expand)Apple Macbook Charging Description
Apple places all the information needed within 100-200px. This allows for the user to read the info without having to scan across the entire screen.

Signal-to-Noise (click to expand)Apple Macbook Screen Description
Every pixel that is not white has relevant information on the screen size and resolution of the new Macbook. White space is used without the addition of ads in the margins.

Hick's Law (click to expand)Instagram Web App
The Instagram Web App has a menu bar that only contains 3 options. Log Out, Profile, and the homepage. This significantly decreases the users decisions on where to go on the website.
